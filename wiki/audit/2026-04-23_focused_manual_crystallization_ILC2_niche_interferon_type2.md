## Why These Four Additional Sources Were Selected

- TLR9/interferon regulation was selected because it directly connects microbial sensing, IFN-gamma, STAT1, and ILC2-driven AHR.
- Lung-gut ILC2 maturation was selected because it adds tissue specialization and inter-organ trafficking to the lung ILC2 model.
- NMU/AREG non-redundant ILC2 function was selected because it anchors neuroimmune regulation and tissue-protective AREG with strong ILC2-specific genetic evidence.
- PAC1/CGRP was selected because it provides a recent inhibitory neuroimmune branch relevant to type 2 airway inflammation.
